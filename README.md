## 介绍

yu是一款快速整理ip c段信息的脚本 , 方便渗透测试人员能够快速整理目标ip范围

## 使用

环境

```python
windows / linux 
python version >= 3.7
```

使用方法

```python
# 查看帮助信息
python3 yu.py -h
```

![images1](./images/1.png)

```
# 指定ip文件 
python3 yu.py -f ip.txt
```
